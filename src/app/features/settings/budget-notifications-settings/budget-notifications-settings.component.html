Cambios en reservas

Activa las notificaciones que se enviarán cuando una reserva se cree, modifique, reagende o cancele. Por defecto, las notificaciones in-app están activas para cliente, profesional y admin; los emails están desactivados hasta que actives el interruptor maestro.

Cliente

El cliente cuya reserva cambia. Si no tiene email, solo se le envía la notificación in-app.

Profesional

El profesional asignado a la reserva. Si no tiene email, solo in-app.

Admin / Owner

Todos los miembros con rol admin, owner o super_admin de la empresa.
